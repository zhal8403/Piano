# ESP32 Wireless Piano

> A 12-key ESP32-powered piano that has two modes one can play notes through a web browser over Wi-Fi, allowing multiple notes to be played simultaneously with a responsive touchscreen interface. the other is a standalone mode 

---

# Features

- **12 responsive piano keys**
- **Wi-Fi hosted intrerface**
- **2 modes Standalone and Wifi**
- **Works on phone, tablets, and computers**

# Assembly

- Print all STL files.
- Install the limit switches into the holes under the key.
- Wire each button to its assigned GPIO pin.
- Connect all switches grounds together.
- Mount the ESP32 inside the case.
- Put the keys in their spots
- Push a piece of Pla filamenmt through the side hole and keys
- Upload the Arduino sketch.
- Connect to the same Wi-Fi network.
- Open the ESP32 IP address in any browser.
- Play!

# How it works

## Wifi-Mode

The ESP32 hosts a webpage over Wi-Fi. When a piano key is pressed, the ESP32 notifies the webpage, which generates audio using the **Web Audio API**. Because the browser creates the sound instead of the ESP32, multiple notes can be played simultaneously without the limitations of a buzzer.

## Standalone-Mode

The ESP32 loops through each key and when a key is pressed, the ESP32 send a tone to the speaker. Though using tone() only allows one note at a time otherwise itll be jumbeled.
